# 선애순 공개 자료 정리

작성일: 2026-04-20

## 정리 원칙

- 이 문서는 `선애순` 관련 공개 자료를 먼저 정리한 뒤, 그 결과를 홈페이지 소개·브랜딩의 근거로 쓰기 위해 작성했다.
- 홈페이지 본문에 직접 반영하는 항목은 로그인 없이 확인 가능한 공개 출처를 우선 기준으로 삼는다.
- 사용자가 직접 제공한 원문이나 접근 제한 페이지는 별도 구역으로 분리해 기록한다.

## 홈페이지 반영 원칙

- 소개 페이지에는 공개 검증 가능한 `저자·서지`, `책`, `논문`, `학위논문`, `공공기관 교육·강연`, `뉴스·언론`만 노출한다.
- 동구육아종합지원센터 2021년 3건은 사용자가 제공한 원문이므로, 필요하면 추후 캡처 또는 공개 링크를 다시 확보한 뒤 공개 목록에 편입한다.
- 브랜딩 문구는 위 자료를 기반으로 `모래놀이심리상담`, `아동·영유아 발달`, `부모교육`, `공공기관 연수`, `현장 상담 경험` 축으로 설계하는 것이 맞다.
